    4. Geração de Código (100% Coberta)

    Construções C Geradas e Testadas

    Mapeamento de Tipos - ✅ inteiro → int - ✅ real → float - ✅ caracter → char - ✅ logico → int (0/1)

    Estruturas de Dados - ✅ Variáveis simples - ✅ Arrays estáticos - ✅ Inicialização de arrays - ✅ Acesso a elementos

    Estruturas de Controle - ✅ se/senao → if/else - ✅ para → for - ✅ enquanto → while - ✅ escolha → switch

    Operadores - ✅ Todos os operadores aritméticos - ✅ Todos os operadores lógicos - ✅ Todos os operadores relacionais - ✅ Operadores bitwise - ✅ Precedência preservada

    Exemplo de Código Gerado

    Entrada Portugol: 

    programa {
    inteiro fatorial(inteiro n) {
        se (n <= 1) {
            retorne 1;
        } senao {
            retorne n * fatorial(n - 1);
        }
    }

    funcao inicio() {
        inteiro num = 5;
        escreva("Fatorial: ", fatorial(num), "\n");
    }
}

    Saída C Gerada: 

    #include <stdio.h>

int fatorial(int n) {
    if (n <= 1) {
        return 1;
    } else {
        return n * fatorial(n - 1);
    }
}

int main() {
    int num = 5;
    printf("Fatorial: %d\n", fatorial(num));
    return 0;
}
